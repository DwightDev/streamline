How to use this editor

Edit the Pricing Master page directly in the browser, save versions, then export clean HTML.

  1. Click any outlined text on the page and type your changes.
  2. Select text, then use Bold, Italic, Bullet list, or the color palette.
  3. Use ☰ Drafts to save a version. Drafts can be loaded, renamed, or deleted.
  4. Use Edit link while your cursor is inside a button/link to change its URL.
  5. Use Preview to temporarily hide the editor toolbar and outlines. Press Esc to return.
  6. Use Download clean HTML when you are ready to export the page without editor controls.
Drafts are saved in this browser using localStorage. They will work tomorrow on the same computer/browser, but they will not automatically sync to another device.
