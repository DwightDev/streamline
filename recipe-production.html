Production Recording

Turn recipe work into accurate inventory updates.

Recipe Production is for recording what was made, which recipe was used, and which ingredient lots were consumed. It keeps production and inventory connected from the mobile workflow.

Create & Produce

Create a recipe, then use it for a production run.

Teams can create a recipe through the Inventory App, select it for production, enter the batch quantity, and confirm the ingredient lots used. Once recorded, the app helps keep ingredient stock and finished goods aligned with the actual run.

  • Create a recipe directly through the Inventory App
  • Select a recipe before starting production
  • Enter batch quantity for the run
  • Confirm ingredient lots used
  • Record finished goods for inventory accuracy

Lot Use & Finished Goods

Keep production records tied to inventory movement.

The feature focuses on the practical production record: what recipe was made, how much was produced, which lots were used, and how inventory should change afterward.

Ingredient lots are not an afterthought.

Because ingredient lots are confirmed during the run, teams can record production in a way that supports inventory accuracy and traceability.

  • Confirm the lots used in production
  • Deplete ingredient stock after the run
  • Keep production activity connected to lot history

Finished goods move back into stock.

Once the production run is recorded, finished goods can be added to inventory so warehouse and production teams are working from the same stock position.

  • Record finished product quantities
  • Update inventory after production
  • Reduce manual follow-up between teams

FAQ

Can users create a recipe through the Inventory App?

Yes. Recipe Production includes the ability to create a recipe through the Inventory App, then use recipe details when recording production activity.

What does Recipe Production record?

It records the selected recipe, batch quantity, ingredient lots used, and finished goods produced so inventory can stay aligned with production.

Does it deplete ingredient stock?

Yes. The Inventory App page context describes Recipe Production as recording production runs and depleting ingredient lots automatically.

How does this support traceability?

By confirming ingredient lots used in a production run, teams can keep the finished batch connected to the ingredients that went into it.
