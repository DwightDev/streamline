Import dashboard

See order import status before the inbox develops a personality.

The Home screen shows imported orders, pending reviews, failed imports, processed orders, scanned emails, last scan, next scan, and recent activity.

  • Track imported, pending review, failed, and processed order emails
  • See last scan and next scan timing
  • Run a manual scan with the Scan Now button
  • Review recent activity without opening five more tabs

Order emails

Review order emails without spelunking through Gmail.

The Emails tab lists order-related emails with sender, subject, preview, timestamp, unread marker, and import status.

  • See processed order emails in a focused list
  • Review sender, subject, preview, and timestamp
  • Keep email-based wholesale order management easier to follow
  • Give customer service a calmer place to start

Customer whitelist

Scan the right senders, not every inbox gremlin.

Customer email whitelist settings help teams control which sender addresses are scanned for orders. Add customers, search the list, and keep order intake focused.

  • Add customer sender addresses for scanning
  • Search whitelisted customers quickly
  • Reduce noise from non-order emails
  • Support cleaner wholesale bakery software workflows

Gmail labels and alerts

Give every order email a name tag. Very polite. Very useful.

Customize Gmail labels and notifications for imported, review-needed, failed, manual, and ignored order emails. Your inbox stays organized, and your team knows what needs attention.

  • Apply labels to processed emails
  • Use color coding for faster status checks
  • Customize label names to match your team language
  • Trigger desktop and in-app notifications for key statuses

Scan controls

Run order intake on a schedule, not vibes.

Set automatic import, choose scan frequency, limit scanning to business hours, scan only unread emails, and use subject filters like “Order” to focus processing.

  • Schedule scans every 10, 15, or 30 minutes, hourly, or daily
  • Scan inbox only, unread only, and avoid Trash or Spam
  • Limit scanning to business hours
  • Use subject filters to focus on order-related emails

Common questions

What does the Streamline Chrome Extension do?

It supports browser-based order import workflows by scanning order emails, showing import status, applying Gmail labels, managing customer sender rules, and sending notifications for key order statuses.

Is it for wholesale bakery software workflows?

Yes. It is useful for wholesale bakery teams that receive customer orders by email and need clearer intake, labels, review alerts, and scan controls.

Does it replace Streamline ERP?

No. The extension is a browser-based helper. Streamline ERP remains the connected system for order management, production, inventory, delivery, invoicing, reporting, and customer management.

Can it reduce tab switching?

Yes. It brings order import status, scan controls, email lists, customer sender rules, labels, and notifications into a compact browser extension view.
